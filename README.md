Este proyecto busca un puntaje diferente a fork o tripadvisor, 
o tambien diferente a google maps, busca un sistema de puntuaje donde el usuario pueda
ver bien si ese restaurante le puede merecer la pena o mas bien no visitarlo.

Para eso el puntuaje el puntaje esta dividido en:

1. la Decoracion del Restaurante
2. Como esta estructurado el menu
3. Como de buena esta la comida
4. El servicio que calidad a tenido
5. El precio es bueno o malo
